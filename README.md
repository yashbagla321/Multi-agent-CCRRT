# Multi-agent-CCRRT
This is an implementation of the Multi-agent Chance-Constrained Rapidly-exploring Random Trees algorithm developed in the paper: https://doi.org/10.1115/DSCC2019-9237

Author: Yash Bagla (yashbagla321@gmail.com)

Please cite this repository and the paper if you're using this resource.

## C++ Implementation

A modern C++17 reimplementation lives in [`cpp/`](cpp/). See:

- [`cpp/README.md`](cpp/README.md) — build and run instructions
- [`cpp/ARCHITECTURE.md`](cpp/ARCHITECTURE.md) — system design and data flow
- [`tools/replay_viewer.html`](tools/replay_viewer.html) — browser replay viewer for trajectories, covariance, replans, obstacles, and collision risk

The legacy Python prototype remains in [`Multiagent CCRRT.py`](Multiagent%20CCRRT.py) for reference.
